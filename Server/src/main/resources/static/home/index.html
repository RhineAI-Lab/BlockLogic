Introduce

简介

快速开发一个属于自己的App吧！

我们是北师大人工智能学院学生为班底的学生创业团队，团队兼有复旦大学、厦门大学开发成员。欢迎在下方联系方式处通过邮箱联系我们。

B-Logic是一款以图形编程为主，涵盖编程基础教学，双向代码对照。让对AI有兴趣的同学可以快速在线学习相关算法。提供概念教学，程序编写及生成辅助，在线模型训练的全流程的平台。

同时他支持，在线快捷开发安卓/鸿蒙应用的网站。当前对标麻省理工大学的AppInventor项目，在各方面已有更出色的表现。
